# Prepress — Preparazione alla Stampa

Script per la preparazione dei file alla produzione.

## Script disponibili

| Script | Funzione |
|--------|----------|
| `Colore_Livello.py` | Assegna i colori ai livelli corrispondenti |
| `Crocini.py` | Inserimento crocini di registro |
| `Formato_Foglio.py` | Messa in tavola su foglio macchina |
| `Prepara_per_Grafica.py` | Preparazione layer e geometrie per ufficio grafico |
| `Quota_Selezione.py` | Quote di una selezione |
