# Utilities — Strumenti di Lavoro

Script di utilità per il lavoro quotidiano in cartotecnica.

## Script disponibili

| Script | Funzione |
|--------|----------|
| `_Quota_Selezione.py` | Quotatura interattiva di geometrie selezionate |
| `_Crocini.py` | Inserimento crocini di registro e stampa |
| `_Colore_Livello.py` | Sposta su livello per Colore |
